TSP Island-GA - C++ (MPI) source
================================
Requires: OpenMPI (mpicxx) and a C++17 compiler.

Build (preferred):
  make                 # builds tsp_island (MPI) + tsp_sequential (baseline)
  make test            # builds and runs the unit tests (no MPI needed)
  make clean

Build (manual, if mpicxx is not on PATH):
  mpicxx -O2 -std=c++17 -o tsp_island     tsp_island.cpp
  mpicxx -O2 -std=c++17 -o tsp_sequential tsp_sequential.cpp

Run (single machine, 4 islands, global-best broadcast):
  mpirun --oversubscribe -np 4 ./tsp_island ../data/cities_50.txt --gens 500 --sync 20

Run on a cluster:
  mpirun --hostfile ../cluster/hosts -np 4 ./tsp_island ../data/cities_50.txt --gens 500 --sync 20

Files:
  ga_core.hpp        GA operators (fitness, OX crossover, mutation, tournament)
  local_search.hpp   2-opt + Or-opt local search (optional memetic step)
  tsp_island.cpp     MPI island-model parallel GA (main deliverable)
  tsp_sequential.cpp single-process baseline
  test_*.cpp         unit tests
